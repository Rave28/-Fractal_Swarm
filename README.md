## üöÄ Quickstart

Transform your standard OS into a Sovereign AI Node in under 3 minutes.

### 1. Clone the Hive

```bash
git clone https://github.com/Rave28/-Fractal_Swarm.git
cd -Fractal_Swarm
```

### 2. Terraform the OS (Windows)

Open an Administrator PowerShell and execute the global setup to bind the Swarm aliases to your system profile:

```powershell
.\setup_global_swarm.ps1
```

### 3. Boot the Oracle

Silently spin up the FastAPI Knowledge Broker, the ChromaDB vector hive, and the Nightcrawler daemon in the background:

```powershell
swarm-boot
```

### 4. Inaugurate a Workspace

Open any blank directory on your machine and initialize it. This injects the .antigravityrules protocol and seals the namespace.

```powershell
cd C:\Projects\my_new_startup
swarm-init .
```

Your local IDE agent is now bound to the Fractal Swarm OODA loop. Drop a prompt and watch the nanobots build.
